Apresenta um menu interativo com as seguintes opções\+:
\begin{DoxyItemize}
\item 1\+: Consultar dados da conta (CPF, nome, saldo total)
\item 2\+: Editar dados da conta (nome e/ou senha)
\item 3\+: Excluir conta permanentemente
\item 0\+: Voltar ao menu principal
\end{DoxyItemize}

O menu é executado em loop até que o usuário escolha voltar (0) ou até que a conta seja excluída (retorna true para forçar logout).

\begin{DoxyNote}{Note}
Se a conta for excluída, o método retorna true para sinalizar que o usuário deve ser deslogado do sistema 
\end{DoxyNote}
