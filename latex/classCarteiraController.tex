\doxysubsection{Detailed Description}
Controlador especializado para operações de carteiras. 

Esta classe é responsável por gerenciar todas as operações relacionadas a carteiras de investimento, incluindo criação, edição, exclusão e consulta.

Implementa o fluxo completo de criação de carteira\+:


\begin{DoxyEnumerate}

\item Perfil\+: deve ser um dos três tipos válidos
\end{DoxyItemize}
