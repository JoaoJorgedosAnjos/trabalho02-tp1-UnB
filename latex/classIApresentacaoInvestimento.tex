\doxysubsection{Detailed Description}
Interface para os serviços de gerenciamento de investimentos na camada de apresentação. 

Esta interface define os contratos para a camada de apresentação relacionada ao gerenciamento de carteiras e ordens de investimento. Responsável por fornecer interface para criação, consulta, edição e exclusão de carteiras e ordens de compra/venda.
