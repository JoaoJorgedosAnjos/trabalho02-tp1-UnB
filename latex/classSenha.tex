\doxysubsection{Detailed Description}
Classe que representa um conjunto de caracteres usado para autenticar a identidade de um usuario no sistema de investimentos. 

\DoxyHorRuler{0}


A classe {\ttfamily \doxylink{classSenha}{Senha}} e utilizada para armazenar e validar uma senha segura composta exatamente por 6 caracteres. O objetivo e garantir a seguranca de autenticacao, assegurando que a senha atenda aos requisitos minimos de complexidade definidos pelo sistema.

Regras de validacao\+:
\begin{DoxyItemize}
\item A senha deve conter exatamente 6 caracteres.
\item Deve incluir pelo menos\+:
\begin{DoxyItemize}
\item 1 letra maiuscula
\item 1 letra minuscula
\item 1 digito numerico
\item 1 caractere especial permitido\+: {\ttfamily \#}, {\ttfamily \$}, {\ttfamily \%}, {\ttfamily \&}
\end{DoxyItemize}
\item Todos os caracteres devem ser distintos (sem repeticao).
\item Apenas caracteres alfanumericos e os especiais acima sao permitidos.
\end{DoxyItemize}

Metodos disponiveis\+:
\begin{DoxyItemize}
\item {\ttfamily set\+Valor}\+: Metodo publico que define a senha apos validacao.
\item {\ttfamily get\+Valor}\+: Metodo publico que retorna a senha armazenada como string.
\end{DoxyItemize}

Em caso de valor invalido, lanca {\ttfamily std\+::invalid\+\_\+argument}.
