\doxysubsection{Detailed Description}
Classe que representa o tipo de investidor de um usuario. 

\DoxyHorRuler{0}


Esta classe e utilizada no sistema de investimentos para indicar o perfil do investidor de um usuario, levando em conta sua disposicao para correr riscos, objetivos financeiros e tempo que pretende deixar o dinheiro investido. Ela permite que recomendacoes e decisoes financeiras sejam personalizadas conforme o grau de tolerancia a riscos e preferencias pessoais.

O perfil do investidor pode ser classificado em uma das seguintes categorias\+:
\begin{DoxyItemize}
\item "{}\+Conservador"{}
\item "{}\+Moderado"{}
\item "{}\+Agressivo"{}
\end{DoxyItemize}

A classe {\ttfamily \doxylink{class_tipo_perfil}{Tipo\+Perfil}} e responsavel por armazenar e validar o valor do perfil atribuido ao investidor, garantindo que ele esteja entre as opcoes validas predefinidas pelo sistema.

Regras de validacao\+:
\begin{DoxyItemize}
\item O valor deve ser exatamente uma das tres strings\+: "{}\+Conservador"{}, "{}\+Moderado"{} ou "{}\+Agressivo"{}.
\end{DoxyItemize}

Metodos disponiveis\+:
\begin{DoxyItemize}
\item {\ttfamily set\+Valor}\+: Metodo publico que define o perfil apos validar.
\item {\ttfamily get\+Valor}\+: Metodo publico que retorna o perfil atualmente armazenado.
\end{DoxyItemize}

Em caso de valor invalido, lanca {\ttfamily std\+::invalid\+\_\+argument}.
