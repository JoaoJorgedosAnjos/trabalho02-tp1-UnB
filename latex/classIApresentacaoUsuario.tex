\doxysubsection{Detailed Description}
Interface para os serviços de gerenciamento de usuário na camada de apresentação. 

Esta interface define os contratos para a camada de apresentação relacionada ao gerenciamento de contas de usuários. Responsável por fornecer interface para cadastro, consulta, edição e exclusão de contas.
