\doxysubsection{Detailed Description}
Gerenciador central de interface do usuário. 

Responsável por controlar o fluxo de navegação entre telas, limpeza automática do console e apresentação consistente das interfaces. Implementa uma máquina de estados para controlar o fluxo da aplicação.

Executa o loop principal do sistema. 

Implementa a máquina de estados principal, controlando o fluxo de navegação entre diferentes telas e funcionalidades.

O loop continua até que tela\+Atual seja definida como SAIR. Cada estado é processado por métodos específicos que podem alterar o estado para controlar o fluxo da aplicação.

Estados suportados\+:
\begin{DoxyItemize}
\item MENU\+\_\+\+INICIAL → LOGIN/\+CADASTRO/\+SAIR
\item LOGIN → MENU\+\_\+\+PRINCIPAL/\+MENU\+\_\+\+INICIAL
\item CADASTRO → MENU\+\_\+\+INICIAL
\item MENU\+\_\+\+PRINCIPAL → GERENCIAR\+\_\+\+CONTA/\+GERENCIAR\+\_\+\+INVESTIMENTOS/\+MENU\+\_\+\+INICIAL
\item GERENCIAR\+\_\+\+CONTA → MENU\+\_\+\+PRINCIPAL/\+MENU\+\_\+\+INICIAL (se conta excluída)
\item GERENCIAR\+\_\+\+INVESTIMENTOS → MENU\+\_\+\+PRINCIPAL
\end{DoxyItemize}

Estados do sistema\+:
\begin{DoxyItemize}
\item {\bfseries{MENU\+\_\+\+INICIAL}}\+: Tela de entrada com opções de login e cadastro
\item {\bfseries{LOGIN}}\+: Processo de autenticação do usuário
\item {\bfseries{CADASTRO}}\+: Processo de criação de nova conta
\item {\bfseries{MENU\+\_\+\+PRINCIPAL}}\+: Menu principal para usuários autenticados
\item {\bfseries{GERENCIAR\+\_\+\+CONTA}}\+: Funcionalidades de gerenciamento de conta
\item {\bfseries{GERENCIAR\+\_\+\+INVESTIMENTOS}}\+: Funcionalidades de investimento
\item {\bfseries{SAIR}}\+: Encerramento do sistema
\end{DoxyItemize}

\begin{DoxyItemize}
\item Sucesso\+: redireciona para MENU\+\_\+\+PRINCIPAL
\item Falha\+: retorna para MENU\+\_\+\+INICIAL
\item Feedback visual para o usuário
\end{DoxyItemize}

Controle de sessão\+:
