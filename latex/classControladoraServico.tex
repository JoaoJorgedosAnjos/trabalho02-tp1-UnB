\doxysubsection{Detailed Description}
Implementação concreta das interfaces de serviço com persistência SQLite. 

Esta classe centraliza a lógica de negócio e utiliza SQLite para persistência, implementando as interfaces \doxylink{classIServicoAutenticacao}{IServico\+Autenticacao}, \doxylink{classIServicoUsuario}{IServico\+Usuario} e \doxylink{classIServicoInvestimento}{IServico\+Investimento}. Atua como camada de serviço no padrão arquitetural MVC, processando regras de negócio e validações antes de persistir os dados. \begin{DoxySeeAlso}{See also}
