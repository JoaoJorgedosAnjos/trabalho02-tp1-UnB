Apresenta interface de login com\+:
\begin{DoxyItemize}
\item Máscara automática para CPF (aceita apenas números)
\item Validação de formato e dígitos verificadores
\item Tratamento de erros com dicas informativas
\item Opção de cancelamento a qualquer momento

Este método implementa uma interface amigável para autenticação de usuários, com as seguintes melhorias de usabilidade\+:
\item Máscara automática para CPF\+: usuário pode digitar apenas números
\item Tratamento de erros claro e informativo
\item Loop de tentativas até entrada válida
\end{DoxyItemize}

O usuário pode digitar o CPF de duas formas\+:
\begin{DoxyEnumerate}
\item Apenas números\+: "{}12345678901"{} (será formatado automaticamente)
\item Já formatado\+: "{}xxx.\+xxx.\+xxx-\/xx"{} (será aceito se válido)
\end{DoxyEnumerate}
