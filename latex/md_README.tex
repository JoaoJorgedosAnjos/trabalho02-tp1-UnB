Sistema de gerenciamento de investimentos desenvolvido em C++ moderno, com interface de usuário via console. O projeto foi construído seguindo uma arquitetura em camadas (Apresentação, Serviço, Persistência) com persistência de dados em SQLite e lógica de negócio desacoplada por meio de interfaces.\hypertarget{md_README_autotoc_md2}{}\doxysection{\texorpdfstring{Características Principais}{Características Principais}}\label{md_README_autotoc_md2}

\begin{DoxyItemize}
\item ✅ {\bfseries{Arquitetura Profissional\+:}} Design em camadas (Apresentação, Serviço, Persistência) com interfaces para desacoplamento, garantindo um código limpo e de fácil manutenção.
\item ✅ {\bfseries{Persistência de Dados\+:}} Uso do SQLite para armazenamento de dados, garantindo que as informações dos usuários, carteiras e ordens persistam entre as sessões.
\item ✅ {\bfseries{Precisão Financeira\+:}} Cálculos monetários realizados com {\ttfamily long long} (representando centavos) para eliminar erros de arredondamento de ponto flutuante.
\item ✅ {\bfseries{Lógica de Negócio Realista\+:}} Criação de ordens com cálculo de preço baseado em dados de um arquivo histórico externo no formato da B3.
\item ✅ {\bfseries{Interface de Usuário Robusta\+:}} Sistema de menus multinível com validação de entrada em tempo real e mensagens de ajuda contextuais.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md3}{}\doxysection{\texorpdfstring{Estrutura do Projeto}{Estrutura do Projeto}}\label{md_README_autotoc_md3}
