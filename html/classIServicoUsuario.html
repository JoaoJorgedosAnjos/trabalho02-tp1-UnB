Detailed Description

Interface para os serviços de gerenciamento de usuário providos pela camada de serviço.

Esta interface define os contratos para a camada de serviço relacionada ao gerenciamento de contas de usuários. Responsável por operações de persistência e validação de dados de usuário.

virtual bool IServicoUsuario::cadastrarConta ( const Conta & conta)
pure virtual

Cadastra uma nova conta de usuário no sistema.

Valida e persiste os dados da conta fornecida no banco de dados, verificando se o CPF não está duplicado e se todos os dados são válidos.

Parameters
[in]contaObjeto Conta contendo os dados do usuário
Returns
true se o cadastro foi realizado com sucesso, false caso contrário
Note
Deve validar todos os campos da conta antes da persistência
Deve verificar se o CPF já não está cadastrado
Deve criptografar a senha antes de armazenar
Deve retornar false se houver erro de validação ou persistência
