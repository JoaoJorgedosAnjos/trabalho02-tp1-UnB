Detailed Description

Interface para os serviços de autenticação na camada de apresentação.

Esta interface define os contratos para a camada de apresentação relacionada à autenticação de usuários no sistema de investimentos. Responsável por gerenciar a interação com o usuário durante o processo de login.

◆ autenticar()

virtual bool IApresentacaoAutenticacao::autenticar ( Ncpf * cpf)
pure virtual

Autentica um usuário no sistema.

Solicita e valida as credenciais do usuário (CPF e senha) através da interface de usuário, delegando a validação para a camada de serviço.

Parameters
[out]cpfPonteiro para armazenar o CPF do usuário autenticado
Returns
true se a autenticação foi bem-sucedida, false caso contrário
Note
O método deve solicitar CPF e senha do usuário via interface
Em caso de sucesso, o CPF é armazenado no parâmetro de saída
Em caso de falha, deve exibir mensagem apropriada ao usuário
