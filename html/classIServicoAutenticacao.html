Detailed Description

Interface para os serviços de autenticação providos pela camada de serviço.

Esta interface define os contratos para a camada de serviço relacionada à autenticação de usuários. Responsável por validar credenciais e gerenciar o processo de autenticação no sistema.

Parameters

[in]senhaSenha do usuário a ser validada
Returns
true se as credenciais são válidas, false caso contrário
Note
