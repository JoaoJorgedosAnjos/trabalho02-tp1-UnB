Detailed Description

Interface para os serviços de gerenciamento de investimentos providos pela camada de serviço.

Esta interface define os contratos para a camada de serviço relacionada ao gerenciamento de carteiras e ordens de investimento. Responsável por operações de persistência e validação de dados de investimento.

Consulta os dados de uma carteira específica.

Busca no banco de dados os dados da carteira associada ao código fornecido, retornando os dados da carteira e o saldo atual calculado.

Parameters
[in]codigoCódigo da carteira a ser consultada
[out]carteiraPonteiro para armazenar os dados da carteira encontrada
[out]saldoPonteiro para armazenar o saldo atual da carteira
Returns
true se a carteira foi encontrada, false caso contrário
Note
Deve validar o formato do código antes da consulta
Deve calcular o saldo baseado nas ordens executadas
Deve retornar false se a carteira não existir
Os parâmetros de saída só devem ser preenchidos se a carteira for encontrada
